# JPD Helper - FPT Japanese Tool 🇯🇵

Extension hỗ trợ học tiếng Nhật trên **jpd.lovable.app**. Tự động tìm và hiện đáp án khi làm bài.

## 🚀 Cài đặt

### Chrome

1. **Tải về**: Nhấn nút `Code` → `Download ZIP`, rồi giải nén. Hoặc clone:
   ```
   git clone https://github.com/quan1706/JPD_hotrohoctap.git
   ```

2. **Mở Chrome** và truy cập:
   ```
   chrome://extensions/
   ```

3. **Bật Developer mode** (góc trên bên phải).

4. **Nhấn "Load unpacked"** → Chọn thư mục vừa tải về.

5. ✅ Done! Extension sẽ tự động chạy khi truy cập `jpd.lovable.app`.

### Microsoft Edge

1. Truy cập `edge://extensions/`
2. Bật **Developer mode**
3. Nhấn **Load unpacked** → chọn thư mục extension
4. ✅ Done!

## ⌨️ Phím tắt

| Phím | Chức năng |
|------|-----------|
| `F1` | Hiện / ẩn bong bóng đáp án |
| `C`  | Copy đáp án vào clipboard |

## 📝 Lưu ý

- Extension chỉ hoạt động trên trang `https://jpd.lovable.app/*`
- Kéo thả bong bóng tới vị trí thuận tiện

## 🔄 Cập nhật

Khi có bản mới:
1. `git pull` (nếu clone) hoặc tải ZIP mới
2. Vào `chrome://extensions/` → nhấn nút 🔄 reload trên extension
